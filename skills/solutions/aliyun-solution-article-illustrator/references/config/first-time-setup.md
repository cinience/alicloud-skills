# First-Time Setup

If no preference file exists, create one of these:

- project-level: `.alicloud-skills/aliyun-solution-article-illustrator/EXTEND.md`
- user-level: `$HOME/.alicloud-skills/aliyun-solution-article-illustrator/EXTEND.md`

The project-level file overrides the user-level file.

Suggested starting template:

```yaml
---
version: 1
watermark:
  enabled: false
  content: ""
  position: bottom-right
  opacity: 0.7
preferred_style:
  name: null
  description: ""
language: null
default_type: null
default_density: null
default_output_dir: null
custom_styles: []
---
```
