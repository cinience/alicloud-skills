# 场景提示词样例

本目录存放典型场景的提示词样例，便于直接复制使用或作为模板改写。
这些样例偏向端到端编排，适合与根目录 README 里的独立技能示例互补使用。

- AI 内容生产流水线：`ai-content-pipeline.md`
- 智能检索/RAG 系统：`rag-pipeline.md`
- 企业客服与呼叫中心：`contact-center.md`
- 数据平台与分析：`data-platform-analytics.md`
- 计算与基础设施运维：`infra-ops.md`
- Serverless 部署与运行：`serverless-devops.md`
- 产品与 API 盘点：`product-api-discovery.md`

建议：

- 需要网络与云资源运维场景时，可在 `infra-ops.md` 的基础上替换为 `aliyun-vpc-manage`、`aliyun-alb-manage`、`aliyun-dns-cli` 等技能组合。
- 需要多媒体流水线时，可把根 README 中的 `aliyun-wan-video`、`aliyun-kling-video`、`aliyun-vidu-video` 示例嵌入 `ai-content-pipeline.md`。
